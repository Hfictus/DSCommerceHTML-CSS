# DSCommerce
## Sobre este repositório:
Neste repositório, implementa-se o projeto DSCommerce a partir de curso da DevSuperior, usando-se HTML e CSS.<br>
No repositório DSCommerce-Frontend, é possível ver o projeto implementado com TypeScript, React, Vite e Yarn, também sendo encontrado o layout e instruções baseadas nos casos de uso considerados.<br>
No repositório DSCommerce-Backend, encontra-se a implementação com Java e Spring Boot, considerando-se três perfis de desenvolvimento.
